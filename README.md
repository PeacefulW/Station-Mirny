# Станция «Мирный» — Прототип (Фаза 0)

2D Top-Down Survival Base-Builder на Godot 4.

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| E | Добыча ресурса перед игроком / загрузить 1 дерево в термосжигатель |
| Tab | Инвентарь и ручной крафт |
| B | Включить/выключить режим строительства |
| ЛКМ (в режиме строительства) | Поставить постройку |
| ПКМ (в режиме строительства) | Снести постройку |
| P | Открыть энергосеть |
| Пробел | Атака |

## Что есть в прототипе

- Движение персонажа (синий квадрат)
- Система кислорода: на улице O₂ расходуется, внутри замкнутой базы — восполняется
- Строительство за `scrap`: стены, аварийная батарея и термосжигатель
- Ручной крафт: железная руда -> железный слиток
- Определение «внутри/снаружи» (flood fill) — замкни стены и окажешься в безопасности
- Энергосистема: герметичная база восстанавливает O₂ только при поданном питании
- Аварийная батарея: тихая, но конечная
- Термосжигатель: работает на древесине, шумит и привлекает врагов
- Враги (красные квадраты) — идут на шум и ломают стены
- Scrap (жёлтые пикапы) — ресурс именно для строительства
- Руда и древесина — ресурсы для крафта и топлива, а не для стройки
- HUD и UI подсказывают, что `scrap` нужен для строительства, а руда для крафта

## Ранний Игровой Цикл

1. Собирай `scrap` рядом со стартом и с врагов.
2. Строй стены и герметизируй маленькое укрытие.
3. Поставь источник энергии:
   аварийную батарею для быстрого старта или термосжигатель для долгой работы.
4. Если используешь термосжигатель, добывай древесину и загружай её клавишей `E`.
5. Внутри герметичной и запитанной базы O₂ восстанавливается.
6. Добывай железную руду и переплавляй её во вкладке крафта (`Tab`).

## Как запустить

1. Открой проект в Godot 4.6+
2. Нажми F5 (Play)

## Структура проекта

```
core/         — ядро: системы, компоненты, сущности
data/         — ресурсы баланса (.tres)
scenes/       — сцены Godot
assets/       — (пока пусто — заглушки в коде)
```
