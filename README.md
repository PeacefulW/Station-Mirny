## Текущее состояние

Проект находится не в ранней фазе "голого прототипа", а в переходе от прототипа к более устойчивому фундаменту систем и документации.

В репозитории уже есть:
- генерация мира
- горы и добыча
- строительство и базовые инженерные системы
- O2 / indoor-outdoor foundations
- save/load
- локализация
- инвентарь и базовый крафт
- активная работа над runtime/performance architecture

## Быстрый старт

1. Открой проект в Godot 4.6+
2. Запусти проект через `F5`

## Базовое управление

| Клавиша | Действие |
|---------|----------|
| `WASD` | Движение |
| `E` | Взаимодействие / добыча |
| `Tab` | Инвентарь / ручной крафт |
| `B` | Режим строительства |
| `ЛКМ` | Поставить постройку в режиме строительства |
| `ПКМ` | Снести постройку в режиме строительства |
| `P` | Энергосеть |
| `Пробел` | Атака |

## Структура проекта

```text
core/         — ядро: autoloads, systems, entities
data/         — Resource-данные и баланс
scenes/       — сцены мира и UI
assets/       — графика, аудио, шейдеры
locale/       — локализация
gdextension/  — native/C++ часть
docs/         — каноническая карта документации
```
