# Инженерные сети — Дизайн-документ

> Утверждено в обсуждении. Принцип: снаружи — трубы (тайл за тайлом), внутри — модули на стенах (один клик).

---

## Общий принцип

Две зоны с разной механикой:

**СНАРУЖИ** (от источника до стены дома): реальные трубы, прокладываются тайл за тайлом (зажатый ЛКМ — ведёшь линию). 10-30 тайлов максимум.

**ВНУТРИ** (от стены до потребителя): автоматика через стенные модули. Один клик на стену — подключил. Оборудование в комнате подключается к сети комнаты автоматически.

---

## 1. Электричество

### Как работает
- Генератор автоматически питает комнату в которой стоит
- Провода внутри НЕ нужны — ток "в стенах" (металлические стены проводят)
- Чтобы передать ток в соседнюю комнату — **распределитель** на стене

### Распределитель (⚡ оранжевый)
- Размер: 1×1, ставится НА стену между комнатами
- Установка: один клик на стену
- Эффект: ток из комнаты A → комната B
- Потери: медный распределитель = -10% мощности, сидеритовый = 0%
- Лимит: без лимита (но потери суммируются)
- Цепочка: Генератор → Комната A → Распределитель → Комната B → Распределитель → Комната C

### Визуал
- Обычный режим: ничего не видно (провода в стенах)
- Overlay (V → электросеть): комнаты подсвечены зелёный/жёлтый/красный по мощности, распределители = точки на стенах с линиями

### Инженерная задача
- Генератор в центре базы = меньше потерь, но шум привлекает врагов в сердце базы
- Генератор на краю = больше потерь, но враги идут на периферию

## 2. Воздух (O₂)

### Как работает
- Компрессор (2×2) очищает воздух в своей комнате автоматически
- Чтобы подать воздух в соседние комнаты — **вентиляционная решётка** на стене

### Вентиляционная решётка (💨 синяя)
- Размер: 1×1, ставится НА стену
- Установка: один клик
- Эффект: O₂ из комнаты компрессора → соседняя комната
- Лимит: **4 решётки на 1 компрессор**
- Потери эффективности: 1-я = 100%, 2-я = 85%, 3-я = 70%, 4-я = 55%
- Потребление: 5 Вт каждая (нужен ток в комнате с решёткой)

### Большая база
- 4 комнаты = 1 компрессор
- 8 комнат = 2 компрессора
- Мотивация расширять энергосистему

### Двери и утечки
- Открытая дверь: утечка O₂ ~10% в секунду
- Закрытая дверь: утечки нет
- Шлюз: обе двери никогда не открыты одновременно = 0% утечки

### Продвинутый вариант
- Решётка + подключение к бойлеру = подаёт ТЁПЛЫЙ воздух (O₂ + обогрев через одну решётку)

## 3. Вода

### Цепочка снаружи
```
[Насос] → труба → [Очиститель] → труба → [Бойлер] → труба → СТЕНА ДОМА
```

### Порядок на трубе имеет значение
- Вода течёт в одном направлении: от насоса к дому
- Что стоит на пути — то обрабатывает воду
- Без очистителя = грязная вода (можно пить, но отравишься)
- Без бойлера = холодная вода (пить можно, батареи не работают)

### Минимальная конфигурация
- Насос → труба → стена дома (грязная холодная вода, но работает)

### Полная конфигурация
- Насос → труба → очиститель → труба → бойлер → труба → стена дома

### Прокладка труб
- Тайл за тайлом, зажатый ЛКМ (как стены)
- Последний тайл трубы рядом со стеной = автоподключение
- Визуал: тонкая линия на земле (снаружи)
- Overlay (V → водопровод): трубы голубым цветом

### Размещение оборудования
- Очиститель и бойлер могут стоять:
  - **Снаружи** (между насосом и домом) — дешевле, но уязвимо для врагов
  - **Внутри дома** (в техническом помещении / пристройке) — защищено стенами
  - **В пристройке** — маленькая комната рядом с основным домом, классический вариант

### Внутри дома (стенные модули)

**Кран (💧 голубой)**
- Размер: 1×1, на стену
- Даёт: доступ к воде в комнате
- Нужен для: кухни, гидропоники, химстанции, ванны/душа
- Лимит: 3 крана на 1 насос

**Батарея / радиатор (🔴 красная)**
- Размер: 1×1, на стену
- Даёт: обогрев комнаты (от горячей воды)
- Нужен: бойлер в цепочке
- Работает БЕЗ электричества (горячая вода от бойлера на биомассе)
- Лимит: 4 батареи на 1 бойлер
- Дешёвый обогрев без тока

## 4. Тепло / обогрев

### Четыре варианта обогрева

| Вариант | Источник | Нужен ток? | Шум | Этап |
|---------|----------|-----------|-----|------|
| Камин | Биомасса (внутри комнаты) | Нет | Нет | Ранний |
| Батарея на стене | Бойлер → горячая вода | Нет | Нет | Ранний-средний |
| Электрообогреватель | Электросеть | Да | Нет | Средний |
| Решётка с подогревом | Компрессор + бойлер | Да (5 Вт) | Нет | Средний |

### Радиус обогрева
- Камин: только своя комната
- Батарея: только своя комната
- Электрообогреватель: только своя комната
- Решётка с подогревом: O₂ + тепло в одну комнату

### Термосжигатель (уже есть)
- Даёт: ток + тепло (побочный эффект)
- Минус: ШУМИТ → привлекает Очистителей

## 6. Overlay-режим (V)

Переключение по кругу: V → электро → воздух → вода → температура → выкл.

| Overlay | Что показывает |
|---------|----------------|
| Электро | Комнаты (зелёный/жёлтый/красный), распределители, генераторы |
| Воздух | Компрессоры, решётки, уровень O₂ по комнатам |
| Вода | Трубы (снаружи), краны, батареи, направление потока |
| Температура | Цвета от синего (холод) до красного (жара) по комнатам |

## 7. Прогрессия

### Ранняя игра (0-5 часов)
- Камин (обогрев + готовка)
- Без водопровода (пьёшь из жилоростов вручную)
- Батарея Ковчега (ток без проводки)
- Нет компрессора (O₂ восстанавливается пока батарея жива)

### Средняя игра (5-15 часов)
- Насос + трубы + очиститель → чистая вода
- Бойлер → горячая вода → батареи → дешёвый обогрев
- Компрессор + решётки → O₂ по комнатам
- Распределители → ток по всей базе
- Электрообогреватель (альтернатива батареям)

### Поздняя игра (15+ часов)
- Сидеритовые распределители (0 потерь)
- Продвинутый компрессор (8 решёток вместо 4)
- Решётки с подогревом (O₂ + тепло через одну)
- Автоматический контроль (термостаты, датчики)
