# Sprint 01 Goal (2 weeks)

## Sprint Objective
Закрыть **Фазу 0** (включая простейший крафт `1 ресурс -> 1 предмет`) и подготовить каркас **Фазы 1** без регрессий core-loop.

## In Scope
1. Рефактор крупных скриптов-оркестраторов (без изменения геймдизайна).
2. Вынесение хардкода баланса в Resource-файлы.
3. Минимальный крафт (v1): хотя бы `iron_ore -> iron_ingot`.
4. Стабильные smoke-сценарии: старт, сбор, строительство, бой, save/load.

## Out of Scope (Freeze)
1. Новые большие фичи вне вертикального среза (данжи, транспорт, мод API).
2. Глобальная переработка UI/арт-пайплайна.
3. Полный контент Фазы 1+.

## Success Criteria
- Пункт Фазы 0 «Простейший крафт» закрыт фактически и проверен.
- Нет критичных регрессий в core-loop.
- В конце спринта есть обновлённая документация статуса фич.
