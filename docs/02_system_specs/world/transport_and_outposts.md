# Transport and Outposts

## Purpose

Transport extends the playable radius of the world and turns remote extraction into an infrastructure problem rather than pure walking time.

## Scope

This spec owns:
- high-level transport ladder
- relationship between vehicles and outposts
- logistics implications of remote expansion

## Transport ladder

Intended order:
- on foot
- exoskeleton-assisted travel
- rover / buggy
- rail wagon logistics
- branch-specific living or armored transport
- interplanetary ship at endgame scale

## Outpost role

Outposts exist to:
- secure distant resources
- provide survival footholds
- reduce return friction
- feed the main base through logistics

## Design rules

- transport should extend strategic reach, not trivialize the planet
- remote infrastructure should require maintenance and defense
- branch-specific transport should express the philosophy of each path

## Acceptance criteria

- new transport changes how the player plans expeditions
- outposts are useful without replacing the main base fantasy
- remote infrastructure invites tradeoffs rather than pure convenience
