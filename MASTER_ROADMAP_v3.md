# MASTER ROADMAP v3 — Станция «Мирный»

> Финальная версия. Все ключевые решения зафиксированы.
> Принцип: **ДВИЖОК → UI → СТРОИТЕЛЬСТВО → МИР → ВЫЖИВАНИЕ → ИНФРАСТРУКТУРА → ФАУНА → БОЙ → ПРОГРЕССИЯ → КОНТЕНТ → ГЛУБИНА → РЕЛИЗ**
> Правило: механика без UI не считается готовой. Никакого контента пока механики не работают.
> Каждый этап обсуждается перед началом.

---

## КЛЮЧЕВЫЕ РЕШЕНИЯ (зафиксированы)

| Решение | Выбор | Обоснование |
|---------|-------|-------------|
| Камера | **Factorio-стиль (oblique 70-75°)** | Квадратная сетка + объёмные спрайты. НЕ изометрия. Стены не загораживают обзор |
| Тайлы | **Квадратная сетка 12px** | Мелкие тайлы → детальное строительство. Стена=1, верстак=3×4, печь=2×3 |
| Z-уровни | **3 этажа: +1, 0, -1** | Чистое переключение: видишь ТОЛЬКО текущий этаж. Вокруг = чернота. Лестница/люк = E |
| Электричество | **Комнатное + распределители на стенах** | Генератор питает комнату. Распределитель = ток в соседнюю. Без проводов внутри |
| Воздух | **Компрессор + вент. решётки (макс 4)** | Решётка на стене = O₂ в соседнюю комнату. 1 компрессор = 4 комнаты |
| Вода | **Трубы снаружи + краны/батареи внутри** | Насос→труба→очиститель→бойлер→стена. Внутри: кран = вода, батарея = тепло |
| Строительство | **Свободное, как Factorio/RimWorld** | Ставишь стену за стеной, формируешь комнату сам. НЕ готовые здания |
| Спрайты | **Квадратные тайлы с объёмными (Factorio-стиль) спрайтами** | Код = квадратная сетка. Визуал = лёгкая перспектива (передняя грань видна) |

## ЭТАП 1. ДВИЖОК — «Мир физически работает»

### 1.1 Тайловая система 12px
- Переход с 32px на 12px (tile_size в WorldGenBalance)
- Пересборка C++ ChunkGenerator под 12px
- Пересборка ChunkManager (terrain + resource atlases под 12px)
- Коллизии: игрок, враги, стены — всё под 12px
- Тестирование производительности (7x больше тайлов на экран)
- Chunk по-прежнему 64×64 тайла (но тайлы мельче → чанк меньше в пикселях)

### 1.2 Z-уровни (3 этажа)
- Chunk хранит 3 слоя: z=+1 (второй этаж), z=0 (поверхность), z=-1 (подвал)
- Переключение: E на лестнице/люке → мгновенный swap видимого слоя
- Видишь ТОЛЬКО текущий этаж. Вокруг подвала = камень + темнота. Вокруг 2-го этажа = чернота
- z=0: генерируется C++ (как сейчас)
- z=-1: генерируется C++ (больше камня, подземные руды: сидерит, пещеры, водоносный слой)
- z=+1: пустой (строится игроком, нужен фундамент из z=0)
- Лестница вверх / Люк вниз — объекты строительства
- Save/load: каждый слой сохраняется отдельно

### 1.3 Камера и управление
- Плавная камера (следование за игроком, сглаживание)
- Зум (колесо мыши — приблизить/отдалить)
- WASD движение, Shift спринт, стамина
- Курсор: контекстный (молоток при строительстве, кирка при добыче, меч при атаке)

### 1.4 Многотайловые объекты
- BuildingData: size_x, size_y (1×1, 2×2, 2×3, 3×4...)
- Размещение: проверка всех занятых тайлов
- Коллизия: один StaticBody на весь объект
- Визуал: один спрайт на весь объект (Factorio-стиль, с передней гранью)

### 1.5 Текстуры и атласы
- Перегенерация terrain_atlas.png под 12px
- Перегенерация resource_atlas.png под 12px
- Все спрайты: квадратные тайлы, но с лёгкой перспективой (объём)
- Стены: видна верхняя грань + тонкая передняя грань (тень)
- Объекты (печь, верстак): видна верхняя часть + передняя стенка

**Smoke test:** Игрок ходит по миру 12px, зум работает, камера плавная. Люк вниз → подвал (камень, руды, темно). Лестница вверх → пустой этаж. Переключение мгновенное.

## ЭТАП 2. UI КАРКАС — «Всё видно, всё понятно»

> ВСЕ экраны и панели. Даже если данных ещё нет — каркас с заглушками.

### 2.1 Главное меню
- Новая игра → экран создания мира
- Продолжить (последний сейв)
- Загрузить → список сейвов
- Настройки
- Выход

### 2.2 HUD (всегда на экране)
- Шкала O₂ (диегетичная — на шлеме)
- Шкала голода (иконка + полоска)
- Шкала жажды (иконка + полоска)
- Индикатор температуры (при угрозе)
- Индикатор токсичности (при угрозе)
- HP (полоска)
- Стамина (полоска, при спринте)
- Время суток + день + сезон
- Компас / маркер базы
- Мини-карта (правый верхний, M — полная)
- Quickbar снизу (слоты 1-5: инструменты/оружие)
- Индикатор текущего этажа (z=0 / z=-1 / z=+1)

### 2.3 Инвентарь (Tab)
- Сетка слотов (основной рюкзак)
- Drag-drop перетаскивание
- Стакование, разделение стака (Shift+клик)
- Вес: текущий / максимум
- **Слоты экипировки** (справа от сетки, все с самого начала):
  - Голова: шлем/маска
  - Тело: скафандр
  - Фильтр: споровый фильтр (расходник)
  - Баллон O₂: текущий баллон
  - Батарея скафандра: заряд (обогрев, фонарь)
  - Броня: защитная пластина
  - Утилита: модуль (ночное видение, сканер, карманы)
  - Инструмент (активный): кирка / оружие / пустые руки
- Слоты пустые с начала — мотивация крафтить
- Тултипы: название, описание, характеристики, прочность

### 2.4 Меню строительства (B)
- Панель снизу экрана (Factorio-стиль)
- **Категории-вкладки:**
  - Стены/Полы
  - Двери/Шлюзы
  - Энергия (генераторы, распределители)
  - Производство (верстаки, печи, станции)
  - Жизнеобеспечение (компрессор, решётки, камин, обогреватель)
  - Водоснабжение (насос, трубы, очиститель, бойлер, краны, батареи)
  - Оборона (турели, заграждения)
  - Лестницы/Люки
  - Декор/Мебель
- Каждое здание: карточка при наведении:
  - Иконка (крупная, объёмная)
  - Название + описание (1-2 предложения)
  - Стоимость (иконки ресурсов × количество)
  - Размер (3×4 тайлов)
  - Потребление/выработка энергии
  - Требования: "Нужна технология: [Металлургия]" (серое если заблокировано)
- Призрак на карте (зелёный=можно, красный=нельзя)
- Зажатый ЛКМ: строить линию стен (drag)
- R — повернуть, ПКМ — отменить/снести
- Заблокированные здания видны но затемнены

### 2.5 Крафт-панель (E рядом со станцией)
- Открывается при взаимодействии со станцией
- Левая часть: рецепты (фильтр по station_type)
- Правая часть: входы → выход, время, описание
- Кнопка "Крафтить" + прогресс-бар
- Заблокированные рецепты видны но затемнены

### 2.6 PDA-наручник (J) TODOO
- Карта (туман войны, маркеры POI, база)
- Статы (характеристики, навыки)
- Дерево дешифровки
- Журнал (лор)
- Настройки базы (позднее)

### 2.7 Overlay-режимы (V)
- V → электросеть → воздуховоды → водопровод → температура → выкл
- Электро: комнаты (зелёный/жёлтый/красный), распределители, генераторы
- Воздух: компрессоры, решётки, уровень O₂ по комнатам
- Вода: трубы (снаружи), краны, батареи, направление потока
- Температура: цвета от синего (холод) до красного (жара)

### 2.8 Экраны
- Экран смерти (статистика + опции)
- Меню паузы (Esc, уже есть)
- Настройки (уже есть)

**Smoke test:** Все экраны открываются. Инвентарь с 8 слотами экипировки. Меню строительства с категориями и карточками. PDA с вкладками. Overlay переключается. Quickbar работает.

## ЭТАП 3. СТРОИТЕЛЬСТВО — «Стены, комнаты, свой дом»

### 3.1 Свободное размещение
- Стена за стеной, формируешь помещение сам (Factorio/RimWorld)
- НЕ готовые здания
- Снос и частичный возврат ресурсов
- R — ротация для несимметричных объектов
- Зажатый ЛКМ — линия стен
- Стоимость из инвентаря игрока

### 3.2 Типы стен
- Каменная стена (1×1, дешёвая, слабая)
- Железная стена (1×1, стандарт)
- Укреплённая стена (1×1, дорогая, прочная)
- Полы (внутри помещений, 1×1)
- Дверь (1×1, проходимая игроком, блокирует фауну, утечка O₂ при открытии)

### 3.3 Комнаты и герметичность
- Flood-fill определение замкнутых комнат
- Визуал: герметичная комната = подсветка пола
- Прорыв: повреждённая стена → споры → биомасса на стенах
- Ремонт стен (E + ресурсы)

### 3.4 Шлюз
- Двойная дверь, цикл продувки 3 сек
- Не пропускает споры (обе двери не открыты одновременно)
- Размер: 1×3 или специальный объект

### 3.5 Лестницы и люки
- Лестница вверх (2×2) — строится на z=0, ведёт на z=+1
- Люк вниз (1×1) — строится на z=0, ведёт на z=-1
- E на лестнице/люке = переход на другой этаж
- На z=+1 нужен пол (иначе — нельзя строить в воздухе)
- На z=-1 нужно копать (кирка по камню, открываешь пространство)

### 3.6 Освещение
- Внутри базы: светло если ток + лампы (1×1, PowerConsumer LOW)
- Без тока: аварийный красный / темнота
- Снаружи ночью: только фонарь игрока (конус света)
- Подвал: всегда нужен фонарь или лампы
- Свет от камина/печи (тёплый, побочный)

**Smoke test:** Строю комнату любой формы. Дверь, шлюз. Комната определяется герметичной. Строю люк → спускаюсь → копаю подвал → ставлю лампу → светло. Лестница → второй этаж → пустой, строю спальню.

## ЭТАП 4. ЖИВОЙ МИР — «Хочется гулять и смотреть»

### 4.1 Все биомы
- Оранжевые равнины (стартовый)
- Споровые леса (опасный, халкит, биомасса)
- Ледяные пустоши (холод, криосталь)
- Вулканические зоны (жара, газы, сидерит)
- Руины Предтеч (поверхностные структуры)
- Плавные переходы между биомами
- Карта температуры + влажности + спор → автовыбор биома

### 4.2 Флора
- Споростволы (грибовидные колонны, анимация ветра, добываемые → биомасса + волокна)
- Коралловые шпили (берега, хрупкие → кремнезём)
- Жилоросты (влажные зоны → вода + волокно)
- Светомхи (пещеры, биолюминесценция)
- Споровые грозди (опасные, лопаются → спорит)
- Мелкая растительность (шейдер ветра)

### 4.3 Атмосфера и погода
- Споровые облака (частицы, плотность по карте)
- Туман, дымка, пыль
- Ветер (направление, сила)
- Погода: ясно, облачно, дождь, шторм, споровый шторм, магнитная буря
- Визуальные эффекты (частицы, шейдеры)
- Геймплейные эффекты: видимость, температура, споры

### 4.4 День/ночь
- Плавные переходы (4 фазы)
- Цветовая палитра по времени
- Фонарь игрока (конус света)
- Свет зданий виден издалека ночью

### 4.5 POI
- Обломки корабля (у спавна, стартовый лут)
- Залежи руд (видимые маркеры на поверхности)
- Руины Предтеч (поверхность + входы в данжи на z=-1)
- Гнёзда фауны
- Геотермальные источники (вулканы)

### 4.6 Автотайлинг
- Переходные тайлы (16 per пара)
- Вода→песок→трава→земля→камень
- Плавные береговые линии

### 4.7 Подземный мир (z=-1 генерация)
- C++ генерирует подземный слой: камень, пещеры (void), руды
- Подземные руды: больше сидерита, меньше меди
- Водоносный слой (подземная вода)
- Данжи Предтеч как структуры на z=-1 под руинами на z=0
- Корни споростволов (биомасса под землёй)

**Smoke test:** Иду от спавна → равнины → споровый лес (мрачно, туман) → пустоши (снег) → вулканы (дым). Ночью темно и страшно. Дождь. Красиво. Под руинами — вход в пещеры (z=-1). Можно гулять час.

## ЭТАП 5. ВЫЖИВАНИЕ — «Каждая секунда на счету»

### 5.1 Системы-шкалы
- O₂: дренаж на улице, восстановление в базе (компрессор)
- Голод: дренаж 1/мин, эффекты при <30% (замедление, урон при <10%)
- Жажда: дренаж 1.5/мин, эффекты при <30%
- Температура тела: биом + время суток + сезон + помещение + одежда
- Токсичность: накопление в споровых зонах, антидоты
- Стамина: спринт, восстановление в покое
- Все параметры в SurvivalBalance .tres
- Все шкалы на HUD (каркас из этапа 2)

### 5.2 Еда и готовка
- Сырые споровые грибы (с споростволов, риск отравления, голод +20%)
- Сырое мясо (дроп с фауны, высокий риск, голод +50%)
- Корневой бульон (варка жилоростов на камине, голод +30%, жажда +20%)
- Приготовленное мясо (камин/печь, безопасно, голод +60%)
- Синтетическая паста (химстанция, мидгейм, голод +80%)
- Гидропоника (постоянный источник, требует: вода + свет + тепло)
- На холоде голод тратится ×1.5 (калории на обогрев)
- В тёплой базе голод тратится ×0.8

### 5.3 Вода
- Сбор из жилоростов (ручной, ранняя игра)
- Насос (автоматический, на источнике, ток)
- Загрязнённая → очиститель → чистая
- Грязная вода: можно пить, но шанс отравления

### 5.4 Температура и обогрев
- Камин (биомасса, без тока, тихий, обогрев + готовка) — ранний
- Термосжигатель (биомасса, ток + тепло, ШУМНЫЙ) — ранний
- Электрообогреватель (ток, тихий) — средний
- Батарея/радиатор на стене (от бойлера, горячая вода, без тока) — средний
- Вент. решётка с подогревом (компрессор + бойлер = O₂ + тепло) — продвинутый
- Радиус обогрева: только своя комната
- Визуал: overlay температуры (V)

### 5.5 Споры
- Фильтр скафандра: износ, замена (слот экипировки из этапа 2)
- Компрессор: фильтрация воздуха в базе (расходные споровые фильтры)
- Антидоты (химстанция)
- Прорыв: повреждённая стена → споры внутри → биомасса на стенах

### 5.6 Смерть и сохранения
- Экран смерти (статистика, опции — каркас из этапа 2)
- Три режима сложности: Инженер (permadeath), Колонист (save on quit), Турист (автосейв)
- Лоровые сохранения: кровь фауны → чекпоинт (мидгейм)

**Smoke test:** Выхожу → O₂ падает → голод тикает → на холоде быстрее. Нашёл споростволы → собрал грибы → вернулся → приготовил на камине → поел. Без камина замерзаю ночью. Всё связано.

## ЭТАП 6. ИНФРАСТРУКТУРА — «Провода, трубы, свет — база оживает»

### 6.1 Электричество (комнатная система)
- Генератор питает свою комнату автоматически
- **Распределитель** (⚡, 1×1 на стене): ток → соседняя комната
- Медный распределитель: -10% мощности
- Сидеритовый распределитель: 0% потерь (мидгейм)
- Overlay (V → электросеть): комнаты зелёный/жёлтый/красный

### 6.2 Воздух (компрессор + решётки)
- Компрессор (2×2): очищает свою комнату
- **Вент. решётка** (💨, 1×1 на стене): O₂ → соседняя комната
- Лимит: 4 решётки на 1 компрессор
- Эффективность: 100% / 85% / 70% / 55%
- Решётка потребляет 5 Вт

### 6.3 Водопровод (трубы снаружи, краны внутри)
- Насос на источнике → труба (тайл за тайлом, снаружи)
- По пути: очиститель (грязная→чистая), бойлер (холодная→горячая)
- Очиститель и бойлер: снаружи ИЛИ в пристройке ИЛИ внутри дома
- Труба втыкается в стену → автоподключение
- **Кран** (💧, 1×1 на стене): вода в комнате (макс 3 на насос)
- **Батарея/радиатор** (🔴, 1×1 на стене): обогрев от горячей воды (макс 4 на бойлер)
- Overlay (V → водопровод): трубы, краны, направление потока

### 6.4 Генераторы
- Батарея Ковчега: конечная, бесшумная (есть)
- Термосжигатель: биомасса, шумный, тепло (есть)
- Споровый ферментатор: бесшумный, нужны фильтры
- Солнечная панель: днём, зависит от сезона
- Электростатический коллектор: пик в шторм
- Кондуит Предтеч: мощный, опасная зона (поздняя игра)

### 6.5 Автодобыча
- Электробур (на руде, авто, шумный, ток)

## ПРАВИЛА

1. Каждый этап обсуждается перед началом — детали, решения, приоритеты

3. Все тексты через Localization.t() — ни одной строки в коде
4. Все данные в .tres — ни одного числа в коде
5. Smoke test в конце каждого этапа — "играю 30 минут без багов"

8. Спрайты: квадратная сетка, объёмный Factorio-стиль (НЕ изометрия)
